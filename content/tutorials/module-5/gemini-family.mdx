## Tier mental model

- **Flash / Flash-Lite** — high QPS, lower cost, great for classification, extraction, chat guardrails.
- **Pro** — heavier reasoning, coding, multi-step tool use when latency allows.
- **Ultra (when offered)** — frontier quality for the hardest evals — premium slot.

Names change with releases — bind integrations to **stable endpoint IDs** where possible.

## Multimodal first-class

Many Gemini models accept **image, audio, video, PDF** parts interleaved with text in the **content** array. Your client code must **MIME-type** payloads and respect per-file size caps.

## Grounding and tools

Google stacks **Search grounding**, **Maps**, **code execution**, etc., as optional flags — feature-gate them in product because billing and latency differ from plain text completion.

<Callout type="tip">
Use **Google AI Studio** for quick probes, then move to **Vertex AI** for enterprise IAM, VPC-SC, and consolidated billing — same models, different control plane.
</Callout>

## Key takeaways

- Pick **Flash vs Pro** from **latency + quality** requirements, not branding.
- Multimodal = more **client-side validation** (size, type, PII redaction).
- Read **quota + region** docs before you architect synchronous user paths.
